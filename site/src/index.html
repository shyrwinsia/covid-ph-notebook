This choropleth map shows COVID-19 cases normalized by population for Admin Level 3 (municipalities and cities) of the Philippines.

This map aims to make patterns on the spread of the disease along with the effects of the community and government efforts apparent. This only displays a "big picture" and does not provide any interpretation.

All data comes from the public domain. Aggregation is done manually and on best-effort. Complete accuracy is not guaranteed. This map may not be always up to date. The author is not liable for any damages that result in the usage of this map.
